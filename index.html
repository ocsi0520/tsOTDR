OTDR to XLSX

Kudos for sid5432 who created jsOTDR and published a partially reverse-engineered SOR file specification.

This site is built upotn sid5432's work and is aim to allow users convert SOR files to xlsx files inside a browser. (You don't upload your file to a server, everything is done inside your browser.)

Steps to do
  1. upload your SOR file (it only gets into the browser, nothing is sent)
  2. click on "convert"
  3. download your xlsx file
